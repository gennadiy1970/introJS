Конкатенация

Оператор + "склеивает" строки


'abc' + 'def' 'abcdef'

Сравнение строк лексиграфическое

Строки сравниваются лексикографически, т.е. посимвольно.


больше код числа, 
    "больше" буква
'a'.charCodeAt(0); 97
'A'.charCodeAt(0); 65
'a' > 'A'; true

// длинна строк не важна
'aBC'  > 'Abcdefg' true
'5'  > '12345'    true

'ё'.charCodeAt(0);  1105
'е'.charCodeAt(0);  1077
'ёж'  > 'ем'; true

'ёж'.localeCompare('ем') -1 
'ем'.localeCompare('ёж') 1   
'ем'.localeCompare('ем') 0

Методы строк

Досуп к символам

'abc'.length 3

'abc'.charAt(2) c

'abc'[2] c

Поиск символа в строке

'abc'.indexOf(b) 1

'abc'.indexOf(f) -1

Получение подстроки

'abcde'.slice(1,4) bcd

'abcde'.slice(2) cde

Изменение регистра

'aBc'.toLowerCase () abc

'abc'.toUpperCase() ABC

Замена подстроки

'abc'.replace('bc','yz'); ayz

CharCode

String.fromCharCode(100) d

'day'.charCodeAt(0) d -> 100

'ёж'.localeCompare('ем') -1

'ёж'.localeCompare('ём') 1
