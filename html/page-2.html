Синтаксис

Си-подобный синтаксис - это приятная новость для тех кто изучал Java или Python

Что нужно помнить:

  1. чувствителен к регистру и использует кодировку символов Unicode UTF-16
  2. инструкции называются statements и разделяются точкой с запятой ;

    Правила вставки точки с запятой

    Точка с запятой ставится автоматически. Но бывают редкии ситуации, когда автоматическое решение может дать неожиданный результат. Поэтому лучше ставит точку с запятой всегда. 

    
пример неоднозначности в коде 
    из-за отсутсвия скобок
    function sum(a, b) {
    return a + b;
  }
каррирование 
  foo(a, b) = foo(a)(b) 
  function getB (b) {
    return function getA (a) {
        return a + b;
    }
  }
проверяем
  getB(2)(3)  
получаем 5. Работает!

будем добавлять 100
  let add100 = getB(100);
  add100(200) 
получим 300
  
  add100   функция ждет 
  второго аргумента в скобках
  (10 + 20)  использовать
  круглые скобки можно всегда
получим 130, JS не посчитал 
  перенос как разделитель
  и не поставил скобки.
