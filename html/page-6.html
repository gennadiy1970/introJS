Strict mode / "use strict";

Зачем?

Режим strict (строгий режим), введенный в ECMAScript 5, позволяет использовать более строгий вариант JavaScript используя измененную семантику.

  1. заменяет исключениями некоторые ошибки
  2. исправляет ошибки, мешающие оптимизации
  3. запрещает использовать некоторые элементы синтаксиса для улучшения языка

Активизация строгого режима

Чтобы активизировать строгий режим для всего скрипта, нужно поместить оператор "use strict"; или 'use strict'; перед всеми остальными операторами скрипта


Синтаксис переключения 
    в строгий режим всего скрипта
"use strict";
var v = "Привет! 
  Я скрипт в строгом режиме!";

Чтобы применить строгий режим только внутри блока, необхожимо использовать функцию в качестве такого блока


function strict() {
    Strict режим РАБОТАЕТ 
        внутри функции
}    
Strict режим НЕ работает 
        вне функции
