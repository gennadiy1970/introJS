Prompt, Alert & Confirm

BOM

Все эти три метода относятся к API браузера, а не являются встроеным свойством языка JS.

Как методы работы с браузером используются крайне редко из-за блокирующего характера

alert('выведет текст')

prompt

Возвращает тип данных: string, введенную пользователем. Если пользователь ничего не ввел, то вернет значение по умолчанию. Если нет значения по умолчанию, то undefined 


const please = 'Type text, please!';
const hello = 'Hello World';                
const userString = prompt(
    please, 
    hello
    );                
const userEmptyString = prompt(
    please
    );

confirm

Возвращает true или false


const ask = 'You ara learn JS?';
const userAnswer = confirm(ask)
