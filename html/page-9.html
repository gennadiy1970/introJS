Приведение к числу

Number.parseInt() - разбирает строковый аргумент и возвращает целое число.


parseInt('200.5px') 
    200
parseInt('20px') 20
parseInt('200.5px', 10) 
    200
parseInt('0x20') 32
parseInt('20',8) 16
parseInt('20',16) 32
parseInt('1000',2) 8

Number.parseFloat() - разбирает строковый аргумент и возвращает число с плавающей запятой


parseFloat('200.5px') 
    200.5
parseFloat('20px')    
    NaN

Проверка на число

Number.isInteger() - определяет, является ли переданное значение целым числом.


Number.isInteger(5);    
    true
Number.isInteger(-5);    
    true
Number.isInteger(5.1);    
    false

Number.isFinite()


isFinite(1/0); false
isFinite(-Infinity); false
isFinite(NaN);  false

Number.isNaN() 


isNaN(NaN);   true
